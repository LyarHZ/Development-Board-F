# Development-Board-F
尺寸完全兼容Robomaster开发板C型，使用了STM32H723VGH6主控，增加了远程调试功能，尽可能多的扩展了串口和IO

## 使用说明
使用立创EDA设计
6层PCB符合嘉立创免费打样规则

## 相关仓库

- [高速无线DAP调试器Lite](https://oshwhub.com/ylj2000/dap_hs_esp_open) ― ? 感谢老哥的无线调试器，如果不需要可以自行引出SWD接口。
